# agentics-glossary

Glossary of Agentics, MCP, and agentic AI terms.

## What you get

- `agentics-glossary` skill — authoritative term definitions
- `/glossary [<term>]` slash command

## Install

```
/plugin marketplace add agenticsorg/claude-plugin
/plugin install agentics-glossary@agenticsorg
```

## Try it

```
/glossary
/glossary MCP
/glossary swarm
```

# agentics-tutorials

Step-by-step tutorials for the Agentics community.

## What you get

- Three tutorial skills: `getting-started`, `build-agent`, `mcp-integration`
- `/tutorial [list|<name>]` slash command
- `agentics-tutor` agent (Sonnet) — patient, hands-on tutor

## Install

Via the `agenticsorg` marketplace:

```
/plugin marketplace add agenticsorg/claude-plugin
/plugin install agentics-tutorials@agenticsorg
```

## Try it

```
/tutorial list
/tutorial getting-started
```

Or summon the tutor agent directly via the Task tool with `subagent_type: agentics-tutor`.
